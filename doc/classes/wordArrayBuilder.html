\wordArrayBuilder

Builds an array via a string buffer

This class is used to concatenate strings, then push the result at the end of the array. Once all relevant strings are pushed, the array can be outputted.

// Example:
$pBuilder = new wordArrayBuilder();
$pBuilder->concatenateBuffer('Sample');
$pBuilder->concatenateBuffer('Text');
$pBuilder->pushBuffer();
$pBuilder->concatenateBuffer('Pushed');
$pBuilder->pushBuffer();
print_r($pBuilder->toArray());

// Results:
Array
(
    [0] => SampleText
    [1] => Pushed
)
